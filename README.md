# 버튼잡기게임(C# 코딩)

## 개요-C# 프로그래밍학습
-핵심기능: 마우스를 가져가 대면 도망가는 버튼을 쫓아 눌러야 하는 게임입니다.

-화면구성: 메인화면,  도망가는 버튼, 다시 시작 버튼(시작시 안보임)

## 구현시어려웠던점

- 버튼의 크기가 줄어드는 기능을 설정했을 때, 자료형이 맞지 않는 두 데이터를 가지고 하려다가 오류가 났었습니다.

그래서 정수형으로 쓰겠다고 명시적 형변환을 해서 해결했습니다.

-효과음을 넣을때 어려웠는데 그래서 soundplayer 명령어를 이용해서 추가했습니다.
